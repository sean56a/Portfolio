01 Overview

This project is a cross-platform mobile application built with Flutter, targeting both Android and iOS from a single codebase, with an emphasis on smooth navigation and a native feel.

02 Problem

Building and maintaining separate native apps for Android and iOS is time-consuming for a solo developer, while many cross-platform tools compromise on performance or UI polish.

03 Features

  • Single codebase deployed to both Android and iOS.
  • Custom widget-based UI matching a consistent design system.
  • Local data persistence for offline usability.
  • Smooth page transitions and gesture-based navigation.

04 Technologies

FlutterDartMaterial DesignLocal storage (Hive/SQLite)

05 Architecture

The app follows a widget-tree structure typical of Flutter, with a clear separation between UI widgets, state management, and a local data layer responsible for persistence — keeping the app functional without a constant network connection.
